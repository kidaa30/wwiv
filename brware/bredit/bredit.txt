INSTALLATION
============

Installation of BREdit is very striaght forward.

1) Create a directory to hold BREdit (i.e. C:\WWIV\BREDIT)
2) Unzip the distribution (or copy BREDIT.*) to the directory
   2a) i.e. PkUnZip BREDIT10.ZIP C:\WWIV\BREDIT
        -or-
   2b) COPY BREDIT* C:\WWIV\BREDIT



CONFIGURATION
=============
